# TreeifyJson

A modern JSON Tree Viewer built with React + Go Fiber.

## Features

- JSON Upload
- Tree Visualization
- Search Nodes
- Highlight Search Results
- JSON Validation
- Fast Parsing
- Responsive UI

## Tech Stack

Frontend:
- React
- Vite
- TailwindCSS
- Axios

Backend:
- Go
- Fiber

## Run Frontend

npm install
npm run dev

## Run Backend

go mod tidy
go run main.go
